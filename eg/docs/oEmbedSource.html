NAME

Noembed::oEmbedSource - a base class for sites with existing oEmbed support

DESCRIPTION

This is the simplest of base classes. It is meant for sites that already support oEmbed. For these sites we can simply proxy requests to their oEmbed endpoint. These classes can skip defining serialize and build_url, but must define oembed_url.

METHODS

oembed_url

Must return a string with the site's oEmbed endpoint. e.g. http://soundcloud.com/oembed
